# pravoshop
 
Модули OpenServer:
- Apache 2.4 + PHP 5.5 + Nginx 1.17 
- PHP 5.6
- MySQL 5.6

При регистрации присутствует согласие с обработкой персональных данных.
Возможно удаление личных данных с сайта
Уведомление об использовании Cookies
